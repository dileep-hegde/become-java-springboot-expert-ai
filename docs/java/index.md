## Learning Path

1. Start with [Core Java](./core-java/index.md) to re-establish language fundamentals.
2. Then read [Object-Oriented Programming](./oops/index.md) and [Java Type System](./java-type-system/index.md) to rebuild the conceptual model most other topics rely on.
3. Continue with [Core APIs](./core-apis/index.md), [Collections Framework](./collections-framework/index.md), and [Exceptions](./exceptions/index.md) for day-to-day Java work.
4. Move into [Functional Programming](./functional-programming/index.md), [Multithreading & Concurrency](./multithreading/index.md), and [I/O & NIO](./io/index.md) once the basics are solid.
5. Finish with [JVM Internals](./jvm-internals/index.md), [Java Modules](./modules/index.md), [Java Evolution](./java-evolution/index.md), and [Design Patterns](./java-design-patterns/index.md) for deeper reasoning and interview depth.
